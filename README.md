# Multi-floor-corporate-network-simulation
Multi-floor corporate network simulation built in Cisco Packet Tracer (3 floors, 9 VLANs/departments). Full IPv4 subnetting, inter-VLAN routing, WAN links, static server IPs (DNS, Mail, TFTP, FTP, Web), and complete router/switch configurations with TFTP backups. Perfect example of enterprise network design with remote IT management.
